So, for the last year and a bit, I’ve been using an Apple Mighty Mouse in combination with a Wacom tablet for day to day work. The tablet’s been great for fine manipulation in things like Flash and Photoshop, but not really a prime device for navigating around the file system, so I resorted to the stock Mighty Mouse for basically everything else.

The Mighty Mouse is terrible in basically every way.

I took the plunge and started looking for a primary mouse that would be worthy of holding for more than 40 hours a week.  If I kept using the Mighty Mouse, I would eventually end up with a horrendous claw for a hand, which, I would have to swap out for a pirate hook.  Not only was the Mighty Mouse not really an ergonomic shape, but the scroll ball is…the worst. Sometimes it’d be great in going one direction, and just fail to function normally in other direction.

The Logitech Performance Mouse MX has been a brilliant replacement. Wireless, but able to be charged while you use it, very good scroll wheel, and most importantly, it’s comfortable to use for long hours. Also, one great thumb button.

For those still gripping the stinking soap bar impersonator, the Mighty Mouse, there are much better options out there by far.
