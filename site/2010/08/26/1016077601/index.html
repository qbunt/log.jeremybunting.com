TinyTLF

Paul Taylor just released a new text layout framework, titled TinyTLF (not in any way related to Adobe’s TLF) that allows for a whole lot more control over things like real CSS in Flash, standards compliant XHTML display, lists and unordered lists, and background colors.

If you’re building something with a need for some things that Adobe’s TLF does not quite provide, give this a shot, it looks brilliant.
